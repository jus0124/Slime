# 유니티를 이용한 2D 횡스크롤 FPS 게임

## 개요
유니티를 활용하여 2D 횡스크롤 FPS 게임을 만들었습니다. 에셋스토어의 무료 에셋과 AI를 이용한 이미지를 에셋으로 활용하였습니다.
